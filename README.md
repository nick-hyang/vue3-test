# Vue 3 + Typescript + Vite

[项目参考]{https://juejin.cn/post/6951649464637636622}

## 初始化项目

```bash
npm init @vitejs/app

✔ Project name: … vue3test
✔ Select a framework: › vue
✔ Select a variant: › vue-ts

cd vue3test

npm run install

npm run dev
```

### 添加常用库

-   vue-router: npm install vue-router@4
-   vuex: npm install vuex@4

### 格式化代码

npx prettier --write .
